Forex Scanner CLI

This repository contains a Python command-line tool that fetches live Forex exchange rates from the Alpha Vantage API.

Quick Start

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export ALPHAVANTAGE_API_KEY=your_key
python forex_scanner.py --pairs EUR/USD USD/JPY

Customize the pairs via the --pairs flag or configure continuous updates with --refresh.
